Other skins are at: http://www.winamp.com/skins/

You can use this program with EVERY(!) Winamp skin that works for Winamp.
If you have downloaded a skin, follow these steps:
1) Change the extention of the skin file to .zip
2) Extract it.
3) Change the extention back to .WSZ

Now specify the path in the Load-procedure of frmMain.
